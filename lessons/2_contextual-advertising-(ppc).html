После проверки, вы получите комментарии по вашей работе.

В конце обучения студенты выполнившие все задания получат отдельный диплом, об успешном окончании курса.

Домашнее задание

Выбрать один продукт (товар или услугу) и для него:

  • Проверить какие объявления размещают конкуренты.
  • Подобрать до 50 ключевых фраз для Поиска.
  • Подобрать до 50 ключевых фраз для Сетей. Это будет РСЯ-кампания для Яндекс.Директ.
  • Подобрать минус-слова, чтобы отсечь нецелевой трафик по найденным ключевым фразам.
  • Для каждой фразы создать отдельную группу и составить 1-2 объявления.
  • Для каждой фразы придумать быстрые ссылки и изображения для РСЯ
  • Прописать ссылки в РК Директа с UTM-метками.
  • Рассчитать максимально допустимые ставки по разным группам ранее собранных ключевых фраз
  • В итоге вы должны получить две кампании: для Поиска и для РСЯ. Результаты выполнения ДЗ предоставить либо в формате XLS-файла со списком ключевых фраз и соответствующих им минус-слов
